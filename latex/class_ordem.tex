\doxysubsection{Detailed Description}
Representa uma ordem de compra ou venda de ativo. 

Esta classe implementa uma ordem de negociação que contém todas as informações necessárias para executar uma operação de compra ou venda de um ativo financeiro, incluindo identificador, código do ativo, data, preço e quantidade.
